# nk-math-portfolio
This repository contains my attempt at solving selected exercises/problems from my independent study in mathematics.
Contents include:
- Measure and Probability problems (On Going)
- Stochastic Calculus problems (On Going)
- Real analysis problems (Finished)

Measure and Probability, and Real analysis problems and note can be found on Dexter Chua's Cambridge Note website.
Stochastic Calculus problems and note are from the Introduction to Stochastic Calculus 2024 Lecture at Nagoya University by Dr. Serge Richard. I choose this lecture note since it introduces the subject from probability theory and measure theory persepctive.
